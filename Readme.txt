A aplicação "ToDoList" tem como objetivo guardar seus eventos de forma segura, e detalhada. Sendo assim, fará você ficar "ligado" nos compromissos. 
O servidor está logado ao banco de dados MYSQL, ou seja, terá que alterar os dados do arquivo "db.js" com sua autenticação e o nome da sua database.

Obs: Para que o servidor rode conforme foi projetado, será necessario instalar as seguintes tecnolgias:

 - Mysql
 - express
 - sequelize
 - Body-Parser
 - Handlebars
